= COW

A link:https://github.com/tronpis/COW-in-C[COW] interpreter written in C

I know the interpreter for this already exists, but I re-wrote it from C++ to C in order to make it a lot more lightweight. I've also provided the binary executables in the link:https://github.com/tronpis/COW-in-Creleases[releases] page. It works just like my link:https://github.com/Amirreza-Ipchi-Haq/BrainF[BrainF] interpreter.
