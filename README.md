# COW
A [COW](https://github.com/tronpis/COW-in-C) interpreter written in C

I know the interpreter for this already exists, but I re-wrote it from C++ to C in order to make it a lot more lightweight. I've also provided the binary executables in the [releases](https://github.com/tronpis/COW-in-Creleases) page. It works just like my [BrainF](https://github.com/Amirreza-Ipchi-Haq/BrainF) interpreter.
